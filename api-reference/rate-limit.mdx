Rate limits are enforced for both Ethereum Mainnet and Holesky Testnet API endpoints.

### Network-Specific Limits

<CardGroup cols={2}>

### Rate Limit Exceeded

If you exceed the rate limit, the API will return a `429 Too Many Requests` response.
